- React Native Reanimated
> Precisa instalar a biblioteca pelo terminal, comando:
> 'npm install react-native-reanimated@2.8.0'
> No arquivo 'babel.config.js', incluir a linha abaixo:
> plugins: [
      'react-native-reanimated/plugin',
  ],

- Lottie - site de animações, muitas são gratuitas.
> lottiefiles.com
> 'npm install lottie-react-native@5'
> Para iOS precisa instalar também o abaixo:
> 'npm install lottie-ios@3'
